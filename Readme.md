#JavaScript Game - Battle For Earth

## Description
This is a turn based game constructed using HTML, SASS, and JavaScript.

## Get my game
 * Clone my repo
 * Run npm install
 * Run bower install
 * Or go to the following to play: http://development.js-game.divshot.io/
